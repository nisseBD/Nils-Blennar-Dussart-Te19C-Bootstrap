Vad är ett Element?

Element i HTML är hemsidans "byggnadsklossar", det vill säga att olika Element gör olika saker t.ex h1 taggen vilket är en av de vanligaste taggarna som gör att man kan skriva rubriker på sin hemsida. Element består alltid av en starttagg och en sluttagg.

Vad är HTML?

"HTML" står för Hyper Text Markup Language och är ett slags märkspråk. HTMLs syfte är att definera information på en hemsida vars språk kommunicerar med webbläsaren så att webbläsaren vet var informationen ska visas på din skärm. HTML är inte ett programmeringsspråk.

Vad är ett ID?

Ett ID är en unik class som ger en rad av kod ett unikt id vilket möjliggör att man kan förändra olika delar av kod utan att andra rader ska påverkas. Man kan döpa ett ID till vad man vill.

Vad är en Class?

En Class är samma som ID fast man kan ge en specifik klass egenskaper till hur många fler rader av kod som man vill. Man kan tänka Classer som efternamn till olika rader kod.

Vad är Meta?

META är en del som inte syns på hemsidan men är mer gömd bakom kod. Man kan hitta META-märken inom HEAD element för att ange information om sidan. Flera söktjänster som google och yahoo använder META som ett sätt att samla information om innehållet och nyckelord på hemsidan.

Vad betyder Responsive?

Att ha en "Responsive hemsida" är att ha en hemsida som anpassar sig efter vilken enhet du har. Hemsidan ska alltså se snygg ut på både en mobil och på en PC etc.

Vad är http?

HTTP hjälper din enhet att kommunicera med servrar så att din webbläsare kan läsa innehållet från olika hemsidor. HTTP står för HyperText Transfer Protocol.

Vad är https?

HTTPS är samma som HTTP bara att HTTPS använder sig av "SSL" vilket är ett slags program som skyddar den information som skickas mellan servern och din webbläsare. Programmet skyddar informationen genom att kryptera data.

Vad är Compressed Data?

Vissa HTML hemsidor innehåller bilder och data som tar väldigt mycket plats vilket gör att det tar lång tid att komma in på hemsidan. Detta löser vi genom att Komprimera data och "Content caching" vilket är en mekanism som ger data till flera olika servrar för att på så sätt försnabba processen.

Vad är CDN?

CDN står för Content Delivery Network och är ett slags nätverk för leverans av olika innehåll. Med ett CDN så blir webbplatsen snabbare och mer användar vänlig då besökaren avlastas från webbplatsens huvudserver.

Vad är Local Bootstrap?

Den lokala bootstrapen tar olika bootstrap exemplar från närvarande data lokalt.

Vad är JQuery?

JQuery är ett Javascript bibliotek och är designat för att t.ex anropa eller modifiera element på hemsidan samt hantera händelser som musklick. JQuery är bra att använda när man vill ha olika funktioner på sin hemsida som man inte vet hur man gör.

Vad är Flexbox?

Att anända Flexbox är ett sätt att göra sin hemsida responsive där man kan göra en specifik layout som 2x2 grid på bilder som sedan blir responsive. Det finns flera olika layout att välja mellan när man använder sig av Flexbox.

Vad är Grid?

Grid är också ett sätt att göra sin hemsida responsive och är väldigt lik Flexbox. Man bör använda Grid om man vill skapa element som är både i en rad och en kolumn och är mindre bra för endimensionella element.

Vad är Code Snippets?

Code Snippets är custom koder som gör olika saker. Man kan t.ex kopiera en "Code Snippet" som gör en färdigställd navbar och liknande vilket gör själva kodningen mycket snabbare.
